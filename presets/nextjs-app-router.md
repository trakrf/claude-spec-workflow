# Stack: Next.js (App Router)

> **Package Manager**: npm
> **Test Runner**: Jest/Vitest
> **Build Tool**: Next.js

## Lint
```bash
npm run lint --fix
```

## Typecheck
```bash
npm run typecheck
```

## Test
```bash
npm test
```

## Build
```bash
npm run build
```

## E2E Tests (Optional)
```bash
# If playwright.config.ts exists
npm run test:e2e
```
